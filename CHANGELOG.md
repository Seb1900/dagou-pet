# 更新日志

本项目按版本记录用户可见的新增、调整和修复内容，版本号遵循语义化版本。

## [未发布]

暂无。

## [0.3.0] - 2026-07-23

### 调整

- 在原有八档音高上加入循环的旋律起伏，让连续输入的音调变化更明显。
- 原有节奏吸附、`da/gou` 音区与衔接保持不变。
- `Esc` 加入默认“叫”按键；从 `0.2.0` 更新时只迁移未自定义的旧默认组合。
- 点击桌宠的 `ei` 互动音更新为淡出更自然的版本。
- 即时模式、长按延音和松键自然尾音保持原有行为。

## [0.2.0] - 2026-07-23

### 新增

- 紧凑的三页签设置面板和应用内“关于”页。
- NSIS 安装版的 GitHub Release 更新状态与下载流程。
- 设置文件版本、备份、旧格式迁移和损坏恢复。
- 隐私、安全、支持、贡献、第三方许可和名称使用说明。
- 可开关的狗叫旋律；关闭后按原调即时响应。
- 大狗交替、按下大松开狗两种声音模式。
- 无级缩放、镜像、反应强度、强制置顶和鼠标穿透。
- 害羞、点击摇尾巴、拖动移动和缩放交互。
- Windows 安装版、免安装版、校验值和打包冒烟测试。
- 关于页的恢复默认设置入口与确认流程。

### 调整

- 右键桌宠直接打开设置。
- 右键打开的设置窗会在桌宠交互区域正上方居中，并保持在当前屏幕工作区内。
- 狗叫旋律和速度移到音量下方；音量上限提高到原来的 160%，新版 50% 对应旧版 80%。
- 关于页、隐私说明和项目文档统一使用 Seb1900 作为开发者与发布主体。
- 托盘菜单只保留常用运行控制。

### 修复

- 改善声音延音、松键收尾、状态渐变和多显示器置顶行为。
- 系统偶尔漏发松键事件时，根据物理按键状态自动结束持续音，同时保留正常无限长按和自然尾音。
