# 大狗桌宠隐私说明

生效日期：2026-07-23

本说明适用于 GitHub 仓库 `Seb1900/dagou-pet` 发布的官方版本。

## 全局按键监听

程序使用 Windows 低级键盘 Hook 接收按下和松开事件，以驱动声音和动作。

- 事件只在本机内存中处理。
- 程序只保留按压编号、按键角色、音高位置和声像，直到该键松开。
- 程序不会保存或上传输入字符、窗口标题、进程名称、剪贴板内容或输入历史。
- 为处理系统漏发松键，程序会在按键被视为按住期间读取该虚拟键是否仍处于按下状态，结果不会写入磁盘。
- “暂停狗叫”会停止响应新按键并结束当前持续音。

## 本地数据

设置保存在 `%APPDATA%\dagou-pet\settings.json`，内容包括音量、声音模式、桌宠大小、镜像、按键绑定和窗口位置。写入时会保留本地备份，损坏文件会在本机重命名保留。

程序不建立账号，不包含广告、分析、崩溃上传或行为追踪。

卸载程序会保留设置文件，方便覆盖安装或重新安装。用户可以自行删除 `%APPDATA%\dagou-pet` 清除设置。

## 网络请求

安装版启动后会延迟检查 GitHub Releases，也可在关于页手动检查。检查和下载使用 HTTPS，并校验 Release 附带的 SHA-256。Portable 只在用户点击检查更新时打开官方发布页。

GitHub 会收到网络通信所需的信息，例如 IP 地址、请求时间和 User-Agent。程序不会把按键事件附加到更新请求。

点击“项目地址”会打开 GitHub；点击“反馈问题”会打开飞书表单。程序不会读取浏览器登录状态或页面内容。
