# 安全政策

## 支持范围

当前只为最新 GitHub Release 提供安全修复。旧测试版可能不会单独维护。

## 私密报告漏洞

请使用 GitHub 的[私密安全报告](https://github.com/Seb1900/dagou-pet/security/advisories/new)。不要在公开 Issue、讨论区或社交平台披露尚未修复的漏洞。

报告中请包含：

- 受影响版本和 Windows 版本
- 可复现步骤和实际影响
- 必要的日志、截图或最小样例
- 已知的缓解方式

请勿在报告中附带真实密码、个人输入内容或其他敏感数据。维护者确认问题后会协调修复与披露时间。

普通 Bug、功能建议和杀毒软件误报请按 [SUPPORT.md](SUPPORT.md) 使用公开 Issue。
